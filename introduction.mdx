## What is OpenHarness?

An **agent harness** is the complete infrastructure that wraps around an LLM to make it a functional agent. The model provides intelligence; the harness provides hands, eyes, memory, and safety boundaries.

OpenHarness is a Python implementation of this pattern, designed to be:

- **Lightweight** — 11,733 lines vs 512,664 in comparable tools (44x lighter)
- **Readable** — 163 files with clear, inspectable architecture
- **Compatible** — works with `anthropics/skills` and `claude-code` plugins
- **Extensible** — add tools, skills, and plugins with minimal boilerplate

## Who is it for?

OpenHarness is built for **researchers, builders, and contributors** who want to:

- Understand how production AI agents work under the hood
- Experiment with tools, skills, and coordination patterns
- Extend the harness with custom plugins and domain knowledge
- Build specialized agents on top of proven architecture
